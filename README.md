### 1. 기술스택
- View: `Thymeleaf`, `JavaScript`
- Server: `Java`, `Spring boot`, `Oauth2`, `Jpa`
- DataBase: `MariaDB`
- Other: `Figma`, `Notion`, `Slack`

### 2. 구현
- fetch 모듈 활용한 동적 페이지
- 외부 API 로그인 (Naver, Google)
- 회원 가입, 매칭 신청 및 확정 CRUD
- 매칭 신청 및 확정 시 멘티 & 멘토에게 메일 보내기
- 관리자 페이지에서 회원 관리
- 채팅 기능 (ChatGPT 활용)
